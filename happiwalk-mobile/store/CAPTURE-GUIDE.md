# Capturas de Pantalla - Guía de Captura

## Pantallas a Capturar (5 imágenes)

### 1. Pantalla Principal (Owner)
- Ejecutar: `npx expo start --ios`
- Mostrar: Botón "Solicitar Paseo" visible
- Capturar con: Cmd+Shift+4

### 2. Selección de Paseador
- Navegar a: Lista de paseadores disponibles
- Mostrar: Fotos, nombres, ratings
- Capturar con: Cmd+Shift+4

### 3. Perfil del Paseador
- Navegar a: Detalle de un paseador
- Mostrar: Badge de verificación, rating, botón "Reservar"
- Capturar con: Cmd+Shift+4

### 4. Chat
- Navegar a: Conversación activa
- Mostrar: Burbujas de mensajes, campo de texto
- Capturar con: Cmd+Shift+4

### 5. Perfil/Configuración
- Navegar a: Perfil del usuario
- Mostrar: Foto, nombre, menú de opciones
- Capturar con: Cmd+Shift+4

## Tamaños Requeridos

| Dispositivo | Resolución |
|-------------|------------|
| iPhone 14 Pro Max | 1290 x 2796 |
| iPhone 14/13 Pro | 1179 x 2556 |
| iPhone 14/13 Mini | 1125 x 2436 |

## Notas
- Usar Simulator o dispositivo físico
- Asegurar que no haya información personal visible
- Guardar como PNG
