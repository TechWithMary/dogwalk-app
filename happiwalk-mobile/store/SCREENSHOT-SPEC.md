# App Store Screenshots Specification

## Required Screens (5 screenshots)

### 1. Home Screen (Owner)
- **Size**: 6.7" (1290 x 2796), 6.5" (1284 x 2778), 5.5" (1242 x 2208)
- **Content**: Owner home with "Solicitar Paseo" button visible
- **Show**: Navigation bar, bottom tabs, booking options

### 2. Walker Selection
- **Content**: List of available walkers with photos, ratings
- **Show**: Walker cards with name, rating, price

### 3. Walker Profile
- **Content**: Walker detail view with photo, bio, reviews
- **Show**: Verification badge, rating stars, "Book now" button

### 4. Chat Screen
- **Content**: Conversation with walker
- **Show**: Message bubbles, input field, send button

### 5. Profile/Settings
- **Content**: User profile with settings options
- **Show**: Profile photo, name, menu options
